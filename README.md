# Kumar Shaikat Bala – Portfolio

Full Stack Web Developer portfolio site, optimized for **GitHub Pages** (`username.github.io`).

## Quick setup

1. **Images**  
   Place your assets in `assets/img/`:
   - `logo.png` – logo for the header (optional; if missing, “KSB” is shown).
   - `profile-img.jpg` – about section photo.
   - `favicon.png` / `apple-touch-icon.png` – optional favicons.

2. **Contact form**  
   - **Default:** Submitting the form opens the user’s mail client (mailto). No backend needed.
   - **Optional:** Sign up at [Formspree](https://formspree.io), create a form, and in `index.html` replace `YOUR_FORM_ID` in the form `action` with your Formspree form ID.

3. **Publish on GitHub Pages**  
   - Push this repo to GitHub.
   - In the repo: **Settings → Pages → Source**: deploy from branch `main` (or `master`), folder **/ (root)**.
   - Your site will be at `https://<username>.github.io/<repo-name>/` or `https://<username>.github.io/` if the repo is named `username.github.io`.

## Tech

- Static HTML, CSS, JS (no build step).
- Bootstrap 5 (grid + utilities), Bootstrap Icons, AOS (scroll animations).
- Works without PHP or a server; contact uses mailto or Formspree.
